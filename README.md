# NoLimits - Documentación del Proyecto

Este repositorio contiene la documentación, gestión y archivos principales del proyecto **NoLimits**, desarrollado durante el semestre.

## Estructura del repositorio

- **Documentación/**  
  Contiene informes, diagramas UML, evidencias, presentación y recursos utilizados durante el desarrollo del proyecto.

- **Producto/**  
  Incluye archivos comprimidos del sistema (backend, frontend) y un archivo de descripción con enlaces a los repositorios.

- **Gestión/**  
  Contiene el documento de registro de definición e identificación del proyecto.

- **Integrantes.txt**  
  Lista de integrantes del equipo.

## Descripción general

NoLimits es una plataforma web que permite visualizar, gestionar y consultar contenido multimedia, integrando información como precios desde distintas plataformas en un solo lugar.

---

## Nota

Algunos elementos del proyecto pueden encontrarse en desarrollo o pendientes, sin afectar las funcionalidades principales del sistema.
